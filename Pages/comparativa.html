Herramientas para la Minería de Datos

RapidMiner

RapidMiner es una plataforma de ciencia de datos que permite la preparación de datos, el aprendizaje automático y el despliegue de modelos. Es fácil de usar y no requiere conocimientos avanzados de programación.

Descargar RapidMiner

KNIME

KNIME es una plataforma de análisis de datos que permite crear flujos de trabajo visuales para la preparación de datos, el análisis y la visualización. Es una herramienta de código abierto y altamente extensible.

Descargar KNIME

Weka

Weka es una colección de algoritmos de aprendizaje automático para tareas de minería de datos. Es una herramienta de código abierto que proporciona una interfaz gráfica de usuario para facilitar su uso.

Descargar Weka

Orange

Orange es una plataforma de análisis de datos que permite la creación de flujos de trabajo visuales para la preparación de datos, el análisis y la visualización. Es una herramienta de código abierto y altamente extensible.

Descargar Orange
